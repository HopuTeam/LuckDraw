# 抽奖系统
## 项目名称
    Luck-Draw(抽奖系统)
## 项目要求
    使用.Net Core MVC
## 前台需求
    登陆注册、验证码(邮件)、找回密码、邮箱验证、多模式抽奖、跨组抽奖
## 后台需求
    暂无
## 项目架构
    架构：.Net Core 3.1 + MVC
    前端：Bootstrap + Layui
## 项目进度
